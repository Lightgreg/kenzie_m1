			<p></p> PARAGRAFOS
<em></em> enfase (transformando em italico)
<strong></strong> enfase com muita importancia (transformando em negrito)
<cite></cite> enfase para titulos(como nomes de livros ou relativos)
<i></i> enfase em italico por simples gosto de textualizaçao ou nomes cientificos.
<b></b> para negrito quando for palavra de ligua difente.
<pre></pre> para texto onde os espaços e organizaçao ja estao prontos (vai mostrar do geito q ta)
<div></div> mesma utilizaçao de <p></p> porem sem uma utilizaçao direta
<spam></spam> mesma utilizaçao de qualquer comando para enfase sem utilizaçao direta
                 simbolos para substituiçao que nao podem ser utilizados durante a codificaçao

                   ALGUMAS FORMAS DE USAR LINKS

    <a href="http://www.google.com" >google.com</a>
    //Link de e-mail que abre o cliente de e-mail padrão dos usuários

    <a href="#page-section" >Link to page section</a>

                             TAGS PARA FORMULARIOS
form = casinha para as informaçoes do formulario, lembrando que o label precisa do atributo for, que vem do atributo id que esta na tag para qual vai ser direcionada.
input = tag com varias funções desde textos a números datas e qualquer outra informação que precise ser identificada.
alguns exemplos =  text; number; date; month; password; radio; time; range; file; color;
placeholder = texto auxiliar dentro das caixas de formularios
radio = bolinha de seleçao
select = criar lista de escolha dentro da pagina ( a select esta diretamente ligada com a OPTION que é para colocar quais sao elas.)
textarea= cols"quantidade de colunas" e rows"de linhas"
